# Los Pollos Restaurants

Food ordering website using Next.js. I used Redux Toolkits for state management and sanity.io for content management.

## Tech Stack

**Tech:** Next.js, Redux, Sanity.io

**UI/UX:** TailwindCSS, Swiper.js

**Animation:** AOS
